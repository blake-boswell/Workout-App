Building a workout app that:
Allows you to log workouts
Snag other people's workouts
Create Exercises, Workouts, Programs, Request to add new Movements

Programs contain:
- Many workouts
- A schedule

Workout contains:
- Many Exercise Instances

Exercise Instances contain:
- Movements
- Exercise
- Sets
- Reps
- Comments

Movements contain:
- Exercises

An Exercise has:
- A description
- The target muscle
- An owner
- A rating
- Comments
- Tips
